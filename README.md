

这是一个简单的农业机械销售管理项目

主要包含模块
      1.设备管理
      2.客户管理

1.设备管理
    - 已出售设备
    - 未出售设备
    - 添加设备
    - 出售设备
    - 设备资料
    - 删除设备
   
2.客户管理
    - 添加客户
    - 已购机客户
    - 未购机客户
    - 删除客户
    - 客户资料


其他说明
1.DATA
    - 没有service交互
    - 数据全部保存在database以及sdcard
    
2.UI
    - 采用activity - fragment 的模式简化ui，提升复用及扩展性
